 # AI Vision Assistant for Visually Impaired People

This repository contains my Bachelor's Thesis project which focuses on 
using Computer Vision and Machine Learning to help visually impaired 
individuals detect and recognize objects in their environment.

## Project Overview

The system uses object detection models to identify real-world objects 
through a camera feed and provide feedback to the user.

The goal is to improve accessibility and assist users in navigating 
their surroundings.

## Technologies Used

- Python
- YOLOv8
- OpenCV
- COCO Dataset
- Machine Learning / Computer Vision

## Features

- Real-time object detection
- Detection of everyday objects
- Assistive feedback system
- Designed for accessibility use cases
